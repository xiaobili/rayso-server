# rayso-server

## 1. 介绍

本文档旨在指导用户搭建和使用基于Node.js的Puppeteer爬虫服务器。Puppeteer是一个Node.js库，提供了一套高级API来控制无头浏览器（如Chromium或Chrome）。它提供了丰富的功能，支持几乎所有的网络爬取任务，如页面抓取、截图、生成PDF、测试和监控等。

## 2. 环境准备

### 2.1 系统要求

- Node.js 12.0.0 或更高版本
- Google Chrome 或 Chromium 浏览器

### 2.2 安装依赖

在项目根目录下执行以下命令以安装所需依赖：

```
bash
npm install
```

## 3. 运行爬虫服务器

```
bash
npm run start
```
